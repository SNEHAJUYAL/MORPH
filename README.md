# MORPH

MORPH is an experiment.

This repository begins as almost nothing - and is allowed to become *anything*.

There is no predefined goal, domain, or roadmap.\
No promise of what MORPH will turn into.\
Only the permission to change.

Anyone can contribute.\
Anything can be added.\
Anything can be transformed.

Over time, MORPH may evolve into:
- software
- art
- documentation
- tools
- research
- chaos
- something we don’t have words for yet

The direction, purpose, and structure of this repository are not decided in advance.\
They emerge from community contributions.

MORPH is not finished.\
MORPH is not stable.\
MORPH is becoming.

If you’re reading this, you are part of that process.

Start anywhere.
